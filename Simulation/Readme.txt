FormulaPi Simulation
Version 1.0.0

To run the software, please use the included Simulation.bat file. 
If you are on a linux/osx system, open a shell/terminal and run:
	java -jar Sim.jar 
	
Requirements:
	Java 1.8 - http://www.oracle.com/technetwork/java/javase/downloads/index.html

Support:
	Windows is the only currently guarenteed supported OS, it has been lightly tested on OSX 10.4, 
	support for OSX and linux is coming soon.
